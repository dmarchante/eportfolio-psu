Lesson 5 Florida Counties Demographics

I had fun with this project as it was an application of the different tools we have been learning throughout the course. I learned that the hardest part is getting started, but after you get started its important to play around with the API and the code to internalize the toolset. However, there is a lot to uncover when it comes to visualizing data from the API. This lesson drove home the concept of how to implement pop-ups effectively. Initially, I was quite confused about the visualization techniques, and frankly I was intimidated. When I started to work with the tools and experimenting it gradually became easier. I ended up using circle for population data to differentiate form the typical polygon convention for county data. For me the biggest thing I learned that it is important to get familiar with the documentation, also if you can take from examples of work that has been done previously that is like what you are trying to achieve utilize it and adapt it. For work I will probably start creating templates for the most commonly used visualizations, so I can easily adapt them to different scenarios, learning from my previous statement regarding using previous examples.

Lesson 6 Fuel Type by State

This assignment was challenging, but the end proved rewarding. I had to create a map based on a given feature layer that rendered different fuel types by color. Additionally, I had to restrict the input by state with a given input utilizing JavaScriptâ€™s prompt method. My initial challenge was figuring out how to save the user input and pass the value into the required field so that only the state was shown. I went ahead and added some error handling as the user input was too open; however, it can be improved further. Now the biggest challenge was having the fuel data show only in the chosen state. Initially, I could only show the different fuel types by color, or the fuel in one state not defined by color. I tried to use the query option built into ArcGIS, but I realized I was overthinking the problem. Ultimately, I was able to render the fuel data by type, and limit the state using the definitionExpression. It took some time, but I finally got it. My main take away in this lesson is that perseverance is key when it comes to programming. I also think my problem-solving skills went up a level.
